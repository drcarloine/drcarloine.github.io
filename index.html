A Journey Through The Ear - 2020


A Journey Through The Ear is a project designed for a module called Perception and Multimedia Computing.
We were given quite an open brief, asked to "Code an interactive sketch or app demonstrating a topic in perception.".
So I decided to focus on how the human ear works.
I created an educational app that offers adults a chance to refresh their knowledge of different parts of the ear.
It has a Tutorial mode that guides the user through the ear,
and a Quiz mode that tests their knowledge.
Made using openFrameworks.
Source code on GitLab: A Journey Through The Ear Code





Orpheus Ka Sangeet - 2019


Orpheus Ka Sangeet, a.k.a. The Live Music Box, is an interactive, user-dependent online experience which takes live webcam input and produces audio and visual outputs.
It was the collaborative final project of mine and Aanandita Gupta in the Introduction to Computational Arts Practice module.
The users are invited to play an instrument (guitar, drums or piano) by moving their body in front of their webcam.
When motion is detected in different sections of the output on the screen a corresponding sound is played.
The user can only see the parts of their body that is moving on screen. The movement is represented in white,
but the user can also choose a different colour.
It was our hope that the discrete representation of the user would encourage them to move more freely.
Created using p5.js, MaxiLib, MaxiExtras, HTML and CSS.
You can read more about it and test it out on this website: Orhpeus Ka Sangeet
Source code on GitLab: Orpheus Ka Sangeet Code
