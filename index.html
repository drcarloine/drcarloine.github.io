A Journey Through The Ear


Project designed for Perception in Multimedia Computing module.
It was an open assignment.
My aim was to create an educational app that would give adults a refresher on their knowledge of different parts of the ear.
Has a Tutorial mode that guides the user through the ear,
and a Quiz mode that tests their knowledge.
Made using openFrameworks.
Source code on GitLab: A Journey Through The Ear
